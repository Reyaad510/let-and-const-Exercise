What is the difference between var and let?

Var can be reassigned, redeclared, mutated and its scope is function scope. While let can be reassigned, can be mutated, but it cannot be redeclared and its scope rules is block scope

What is the difference between var and const?

Var can be reassigned, redeclared, mutated and its scope is function scope. While const cannot be reassigned, it cannot be redeclared, it can be mutated, and it's scope is block scope

What is the difference between let and const?

let can be reassigned, can be mutated, but it cannot be redeclared and its scope rules is block scope. While const cannot be reassigned, it cannot be redeclared, it can be mutated, and it's scope is block scope

What is hoisting

Hoisting in Javascript is a default behavior of moving all the declarations at the top of the scope before code execution. This can cause bugs and errors with the "var" keyword
